# 上线前联调回归清单

本文档用于在切真实短信配置、准备上线前，对账号体系做最后一轮回归检查。

## 一、基础配置检查

### 1. 关闭短信 mock

生产环境需确认：

```env
TENCENT_SMS_MOCK_ENABLED=false
TENCENT_SMS_EXPOSE_MOCK_CODE=false
```

### 2. 腾讯云短信配置完整

必须补齐：

- `TENCENT_SMS_SECRET_ID`
- `TENCENT_SMS_SECRET_KEY`
- `TENCENT_SMS_REGION`
- `TENCENT_SMS_APP_ID`
- `TENCENT_SMS_SIGN_NAME`
- `TENCENT_SMS_LOGIN_TEMPLATE_ID`
- `TENCENT_SMS_BIND_TEMPLATE_ID`

### 3. 短信模板变量检查

当前发送逻辑默认传两个模板变量：

1. 验证码
2. 过期分钟数

确认腾讯云模板变量个数与此一致。

## 二、账号主链路回归

### 场景 1：微信首次登录

预期：

- 新建账号
- 写入 `wechat_openid` / `wechat_unionid`
- 返回 token

### 场景 2：手机号首次注册/登录

预期：

- 发送真实短信成功
- 验证码正确时自动创建账号
- 写入 `phone` 身份
- `user_profiles.phone` 不为空

### 场景 3：手机号二次登录

预期：

- 不新建账号
- 返回同一个 `user_id`

### 场景 4：账号密码登录

预期：

- 已设置用户名密码的用户可以稳定登录
- 返回同一个 `user_id`

## 三、绑定与并账号回归

### 场景 5：已登录用户绑定未占用手机号

预期：

- 手机号绑定成功
- `user_profiles.phone` 更新
- `user_identities` 写入手机号身份

### 场景 6：已登录用户绑定已被占用手机号

预期：

- 返回 `merged_to_user_id`
- 当前账号自动并入主账号
- 主账号保留手机号身份

## 四、资产业务迁移回归

并账号后重点核验：

- `users.stones`
- `user_orders`
- `stone_records`
- `ai_tasks`
- `ai_video_tasks`
- `templates.creator_user_id`
- `template_comments`
- `template_shares`
- `template_likes`
- `template_unlocks`
- `designer_follows`
- `designer_reviews`
- `invite_relations`
- `user_memberships`
- `user_invite_codes`
- `user_profiles`
- `certification_applications`
- `code_sessions`
- `user_identities`

### 检查规则

1. 主账号余额 = 主账号 + 源账号
2. 源账号余额清零
3. 主账号保留更强 `can_withdraw`
4. 主账号保留有效密码
5. 点赞 / 解锁 / 关注关系不能重复
6. 邀请关系不能出现自邀请
7. 会员权益保留更优一份
8. 登录态最终落到主账号

## 五、前端状态回归

### 登录页

检查：

- 微信登录可用
- 手机号验证码登录可用
- 账号密码登录可用
- 登录后本地缓存包含：
  - `id`
  - `username`
  - `phone`
  - `hasPassword`

### 设置页

检查：

- 显示手机号绑定状态
- 显示密码设置状态

### 个人信息页

检查：

- 进入时能正确加载手机号和密码状态
- 绑定手机号成功后页面即时刷新
- 并账号后本地缓存里的 `id` 会更新到主账号

## 六、下载权限回归

### 模板下载

预期：

- 手机号验证通过后，模板详情里的下载权限正常放行

### AI 原图 / 视频下载

预期：

- 手机号验证通过后，AI 图片原图 / 视频下载接口正常放行

## 七、上线前最终检查

1. `go test ./model ./component ./processor`
2. 小程序真机联调：
   - 微信登录
   - 手机号验证码登录
   - 账号密码登录
   - 手机号绑定
   - 自动并账号
3. 管理后台抽查：
   - 用户列表手机号显示
   - 用户详情手机号显示
   - 设计师详情手机号显示
4. 真实腾讯云短信到达率验证
5. 生产环境关闭 mock
