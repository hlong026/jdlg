将支持中文的字体文件放在此目录，水印中文即可正常显示。

支持格式：TTF 或 OTF（思源黑体等多为 OTF）
文件名任选其一：default.ttf 或 default.otf

推荐字体（任选其一，下载后重命名为 default.ttf 或 default.otf）：
- 思源黑体（OTF）：https://github.com/adobe-fonts/source-han-sans/releases
- 文泉驿微米黑（TTF）：https://sourceforge.net/projects/wqy/files/wqy-microhei/
- Noto Sans SC（TTF/OTF）：https://fonts.google.com/noto/specimen/Noto+Sans+SC

若未放置字体，水印会使用系统默认点阵字体，中文可能显示为方框或乱码。
